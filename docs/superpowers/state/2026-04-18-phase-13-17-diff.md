## 新增 CLI（`we-together <subcommand>`）

- `onboard --dry-run` — 5 步新用户引导
- `eval-relation --baseline <path>` — 关系推理 eval + 回归门禁
- `timeline --person-id <id>` — persona_history + active_relations + recent_events
- `relation-timeline --relation-id <id>` — strength 时序
- `what-if --scene-id <id> --hypothesis "..."` — 社会推演 teaser

## 结论

v0.9.0 让 `we together` 完成三个大跃迁：

1. **产品化**：从 SDK → 可分发产品（pip / Docker / 两个 demo）
2. **时间维度**：从"当下切片" → "可回望 + 可前瞻"
3. **质量可测**：从"跑得过" → "跑得对"（groundtruth + regression 门禁）

下一阶段（Phase 18+）重点候选：多模态 importer 深化、社会模拟 SM-2/3/4/5、完整多租户 + rbac 集成。
