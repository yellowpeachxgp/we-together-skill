## 不变式增量

**#23** 扩展点必须通过 plugin registry 注册；核心代码不得为特定 importer/provider/service 硬编。

**#24** 时间敏感服务必须读 `graph_clock.now()` 优先，`datetime.now()` 仅限核心内核。

**#25** 任何跨进程 / 跨图谱出口（联邦 / 导出）必须支持 PII 脱敏与 visibility 过滤。
