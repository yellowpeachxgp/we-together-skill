# Code of Conduct

## 我们的承诺

为了营造开放、欢迎所有人的环境，我们作为贡献者和维护者承诺：不论年龄、体型、残疾、种族、性别认同和表达、经验水平、国籍、外貌、种族、宗教或性取向，参与项目的经历都应**无骚扰**。

## 我们的准则

有助于营造积极环境的行为：
- 使用友好、包容的语言
- 尊重不同观点和经验
- 优雅接受建设性批评
- 关注社区利益
- 对其他成员表示同理

不可接受的行为：
- 性化语言或图像、不受欢迎的性关注
- 挑衅、侮辱 / 贬损评论、人身或政治攻击
- 公开或私下骚扰
- 未经许可公开他人私密信息（住址、电子邮件等）
- 其他职业场合不适当的行为

## 责任范围

维护者负责澄清可接受行为的标准，且应对被判定为不适当的行为采取公正、恰当的纠正措施。

## 范围

本准则适用于项目空间与公共场合。

## 执行

报告滥用、骚扰等行为请发邮件至维护者（见 [README](README.md)）。
所有投诉都会被审查和调查，并在必要时予以响应。

## 归属

本准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) 2.1 版。
